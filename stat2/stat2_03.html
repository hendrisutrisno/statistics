3. Independence of Random Variables

Overview and Motivation

In earlier sections, we introduced joint probability distributions to describe how two random variables \(X\) and \(Y\) behave together, and marginal distributions to summarize each variable by itself.

This section focuses on a key structural property of joint behavior: independence.

Independence formalizes the idea that learning the value of one variable does not provide information about the other variable. When independence holds, probabilities involving both variables become much easier to compute because the joint distribution can be built directly from the marginals.

Definition and Equivalent Characterizations

Let \(X\) and \(Y\) be random variables with joint pmf \(f_{XY}(x,y)\) (discrete case) and marginal pmfs \(f_X(x)\) and \(f_Y(y)\). Conditional pmfs are denoted by \(f_{Y\mid X}(y\mid x)\) and \(f_{X\mid Y}(x\mid y)\).

Independence

For random variables \(X\) and \(Y\), if any one of the following properties is true, then the others are also true, and \(X\) and \(Y\) are independent.

  1. Factorization of the joint distribution

\[f_{XY}(x,y) = f_X(x)\,f_Y(y) \quad \text{for all } x \text{ and } y.\]

  1. Conditioning on \(X\) does not change the distribution of \(Y\)

\[f_{Y\mid X}(y\mid x) = f_Y(y) \quad \text{for all } x,y \text{ with } f_X(x) > 0.\]

  1. Conditioning on \(Y\) does not change the distribution of \(X\)

\[f_{X\mid Y}(x\mid y) = f_X(x) \quad \text{for all } x,y \text{ with } f_Y(y) > 0.\]

  1. Event-level factorization

\[P(X\in A,\, Y\in B) = P(X\in A)\,P(Y\in B)\]

for any sets \(A\) and \(B\) in the ranges of \(X\) and \(Y\).

Each statement expresses the same idea in a different language:

  • “1” is the most direct statement in terms of pmfs.

  • “2” and “3” state independence as “knowing one does not change the other.”

  • “4” states independence using general events (subsets of possible values).

Illustrative Example (Discrete Case)

In a remote environmental monitoring station, researchers study whether manual data entry issues are related to automated data capture outcomes. Suppose:

  • \(X\) = number of minor recording errors in a quality log during a shift, with \(x\in\{0,1,2\}\).

  • \(Y\) = number of sensor snapshots saved during the same shift, with \(y\in\{1,2\}\).

The joint pmf \(f_{XY}(x,y)\) summarizes how the paired outcomes \((X,Y)\) occur across shifts:

\(f_{XY}(x,y)\)

\(x=0\)

\(x=1\)

\(x=2\)

\(y=1\)

0.24

0.08

0.08

\(y=2\)

0.36

0.12

0.12

The main question is whether these variables are independent. In practical terms:

  • Do shifts with more recording errors tend to have a different snapshot pattern?

  • Or does the snapshot process behave the same regardless of the number of errors?

To answer this, we verify independence using the standard equivalent conditions:

  1. Compute the marginal pmfs \(f_X(x)\) and \(f_Y(y)\).

  2. Check whether \(f_{XY}(x,y)=f_X(x)f_Y(y)\) for all \((x,y)\).

  3. Check whether \(f_{Y\mid X}(y\mid x)=f_Y(y)\) whenever \(f_X(x)>0\).

  4. Check whether \(f_{X\mid Y}(x\mid y)=f_X(x)\) whenever \(f_Y(y)>0\).

  5. Illustrate event-level factorization \(P(X\in A, Y\in B)=P(X\in A)P(Y\in B)\) for selected sets \(A\) and \(B\).

Step 1: Compute the marginal distributions

Marginal pmf of \(X\) (sum down each column):

\[f_X(0)=0.24+0.36=0.60,\quad f_X(1)=0.08+0.12=0.20,\quad f_X(2)=0.08+0.12=0.20.\]

Marginal pmf of \(Y\) (sum across each row):

\[f_Y(1)=0.24+0.08+0.08=0.40,\quad f_Y(2)=0.36+0.12+0.12=0.60.\]

Step 2: Verify condition (1) (factorization)

We test whether \(f_{XY}(x,y)=f_X(x)f_Y(y)\) for all \((x,y)\).

For \((x=1,y=2)\):

\[f_{XY}(1,2)=0.12, \qquad f_X(1)f_Y(2)=0.20\times 0.60=0.12.\]

For \((x=0,y=1)\):

\[f_{XY}(0,1)=0.24, \qquad f_X(0)f_Y(1)=0.60\times 0.40=0.24.\]

The same equality holds for the remaining cells. Therefore, condition (1) is satisfied and \(X\) and \(Y\) are independent.

Step 3: Verify condition (2) (conditioning on \(X\))

By definition,

\[f_{Y\mid X}(y\mid x)=\frac{f_{XY}(x,y)}{f_X(x)} \quad \text{for } f_X(x)>0.\]

Compute \(f_{Y\mid X}(y\mid 1)\):

\[f_{Y\mid X}(1\mid 1)=\frac{f_{XY}(1,1)}{f_X(1)}=\frac{0.08}{0.20}=0.40,\]
\[f_{Y\mid X}(2\mid 1)=\frac{f_{XY}(1,2)}{f_X(1)}=\frac{0.12}{0.20}=0.60.\]

These values match \(f_Y(1)=0.40\) and \(f_Y(2)=0.60\). The same match occurs for \(X=0\) and \(X=2\). Therefore, condition (2) holds.

Step 4: Verify condition (3) (conditioning on \(Y\))

Compute \(f_{X\mid Y}(x\mid 1)\):

\[f_{X\mid Y}(0\mid 1)=\frac{0.24}{0.40}=0.60,\quad f_{X\mid Y}(1\mid 1)=\frac{0.08}{0.40}=0.20,\quad f_{X\mid Y}(2\mid 1)=\frac{0.08}{0.40}=0.20.\]

These values match \(f_X(0)=0.60\), \(f_X(1)=0.20\), and \(f_X(2)=0.20\). The same match holds for \(Y=2\). Therefore, condition (3) holds.

Step 5: Verify condition (4) (event-level factorization)

Choose \(A=\{1,2\}\) and \(B=\{2\}\).

First compute the joint event probability:

\[P(X\in\{1,2\},Y=2)=f_{XY}(1,2)+f_{XY}(2,2)=0.12+0.12=0.24.\]

Now compute the marginal event probabilities:

\[P(X\in\{1,2\})=f_X(1)+f_X(2)=0.20+0.20=0.40,\]
\[P(Y=2)=f_Y(2)=0.60.\]

Their product is:

\[P(X\in\{1,2\})P(Y=2)=0.40\times 0.60=0.24.\]

The values agree, illustrating condition (4).

Rectangular Range for \((X,Y)\)

Let \(D\) denote the set of points \((x,y)\) that receive positive probability under \(f_{XY}(x,y)\).

  • If \(D\) is not rectangular, then \(X\) and \(Y\) are not independent.

  • If \(D\) is rectangular, independence is possible but not guaranteed. At least one of the conditions in the definition must still be checked.

Why non-rectangular support implies dependence

If \(D\) is not rectangular, then for some values of \(x\), certain values of \(y\) are impossible. In that case, knowing \(X=x\) changes the set of \(Y\) values that can occur with positive probability. This means \(Y\) depends on \(X\), so independence cannot hold.

Example (non-rectangular support)

Let \(X\in\{0,1,2\}\) and \(Y\in\{0,1,2\}\). Consider the joint support

\[D = \{(x,y): y \le x\}.\]

This means:

  • If \(X=0\), then \(Y\) must be \(0\).

  • If \(X=1\), then \(Y\) can be \(0\) or \(1\).

  • If \(X=2\), then \(Y\) can be \(0,1,2\).

Here, knowing \(X\) restricts the possible values of \(Y\). Therefore, \(X\) and \(Y\) cannot be independent.

Example (rectangular support, but independence not guaranteed)

Now suppose the support is the full rectangle:

\[D = \{0,1,2\}\times\{0,1,2\}.\]

In this case, all pairs are possible, so independence is possible. However, independence may still fail if the probabilities do not factorize.

For instance, if higher probability is concentrated near the diagonal \(x=y\), then large values of \(X\) tend to occur with large values of \(Y\), which indicates dependence even though the support is rectangular.

Common Errors and How to Avoid Them

  1. Checking only one pair \((x,y)\). Independence requires the defining equality to hold for all \(x\) and \(y\).

  2. Assuming marginals imply independence. Marginal distributions always exist. Independence is a special structure of the joint distribution.

  3. Confusing “rectangular support” with independence. Rectangular support is only a necessary condition. You still must verify one of the independence properties.

Summary: What Independence Allows

If \(X\) and \(Y\) are independent, then:

  • The joint distribution can be written as a product of marginals.

  • Conditioning on one variable does not change the distribution of the other.

  • Probabilities for combined events can be computed using only marginal distributions.

Marginal distributions describe how variables behave overall, but independence is a statement about whether the variables are connected in the joint structure.
